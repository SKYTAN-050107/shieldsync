# Logo Files Needed

Place your ShieldSync logo files in this directory:

## Required Files:

1. **shieldsync-logo.svg** - Main logo (horizontal layout recommended)
   - SVG format for scalability
   - Include text "ShieldSync" if possible

2. **shieldsync-logo.png** - PNG backup
   - Minimum size: 512x512px
   - Transparent background recommended

3. **shieldsync-icon.svg** - Icon/badge only (square, no text)
   - Use for social media, app icons
   - Square aspect ratio

4. **favicon.ico** - Browser favicon
   - Multi-size: 16x16, 32x32, 48x48
   - Can be generated from icon.svg using online tools

## Design Guidelines:

- **Primary Color**: #DC2626 (Danger Red) - for emergency theme
- **Secondary Color**: #1D4ED8 (Trust Blue) - for safety/trust
- **Style**: Modern, clean, safety-focused
- **Icon**: Consider shield + sync/connection symbol

## Tools to Create Logos:

- Figma (free): https://figma.com
- Canva (free): https://canva.com
- Logo generators: https://looka.com, https://brandmark.io
- Favicon generator: https://realfavicongenerator.net
